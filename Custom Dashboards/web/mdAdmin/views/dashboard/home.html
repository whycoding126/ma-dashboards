Welcome to Mango's Custom Dashboards Module

Custom Dashboards are a powerful way to create dynamic visualization screens for your customers. They are designed to both impress and offer superior functionality and performance. The development environment is designed for rapid development and deployment.

You can explore the examples and tutorials with the menu on the left. Start off by fiddling with the live examples, you can then copy and paste mark-up into the play area and save it. Once you have some working dashboard mark-up, go ahead and create a dashboard.

For tips on where to find and edit the files see our online documentation.

To get help or ask questions see our forum.

Please note that this module and the examples are still under development. Check for updates regularly as things new features may be released regularly.

Have fun!
