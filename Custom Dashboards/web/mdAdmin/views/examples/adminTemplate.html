This is a complete Material Design admin template, similar to these documentation pages, with a menu and toolbar which you can customise.

Copy the entire ${MA_HOME}/web/modules/dashboards/web/adminTemplate folder into the ${MA_HOME}/overrides/web/modules/dashboards/web directory, give it a new name then customize the menu and other behavior by editing app.js (shown below).

You will need to change the require line at the bottom of index.html to suit the name of the folder you create in overrides.

You can add your own views, controllers, directives and filters. You may also want to define some of the constants shown on the extend app page
