Adaptive Layouts

We have added several example layouts that demonstrate the power of Angular Material's Adaptive Layout API. With no need for complex CSS media queries, and using just simple HTML properties, you can create advanced flexbox layouts. You can use these as starting points for a creating an adaptive layout within your pages.
