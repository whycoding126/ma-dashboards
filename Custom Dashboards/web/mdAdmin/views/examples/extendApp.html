You can extend the AngularJS application and add your own directives, filters and functions to the root scope. You will need to change the require line at the bottom of the file to suit the name of the folder you create in overrides.

The JavaScript also contains a commented section that shows how to auto-login before bootstrapping the application.

Some constants that you may want to define on your app are:

  • mangoBaseUrl – define the Mango base URL if you are displaying data from a remote Mango instance, i.e. not localhost
  • mangoTimeout – define REST API call timeout in ms
  • mangoWatchdogTimeout – watchdog timeout in ms
  • mangoReconnectDelay – time in ms before trying a reconnect if watchdog fails
  • mangoDateFormats – default Moment.js date formats used by the app
  • maDashboardsInsertCss – if false, directives don't automatically insert CSS into the page
