This example demonstrates using the <ma-clock> directive to display an analog style clock. The timezone attribute can be set to somewhere besides the broswer timezone.

{{time|moment:'format':'ll LTS Z'}}
{{time|moment:'tz':user.getTimezone()|moment:'format':'ll LTS Z z'}}
{{time|moment:'tz':'Australia/Sydney'|moment:'format':'ll LTS Z z'}}
{{time|moment:'tz':'America/Denver'|moment:'format':'ll LTS Z z'}}
