This example demonstrates using the <ma-clock> directive to display an analog style clock. By default dates and times are formatted as the server or user timezone but you can set the timezone attribute to display the time in a specific timezone.

See API Documentation for <ma-clock>.

{{time|moment:'tz':browserTz|moment:'format':'ll LTS Z'}}
{{time|moment:'format':'ll LTS Z z'}}
{{time|moment:'tz':'Australia/Sydney'|moment:'format':'ll LTS Z z'}}
{{time|moment:'tz':'America/Denver'|moment:'format':'ll LTS Z z'}}
