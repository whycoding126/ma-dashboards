The Mango v3 Dashboards module uses AngularJS and AngularJS Material Design to make building dashboards easy using simple HTML-like markup. The aim is that you should not have to write any JavaScript. This page will give you a basic introduction to using AngularJS in your markup, please see W3 Schools for more basic examples.

For a more advanced javascript tutorial on Angular Material specifically, check out this free Video Training.

Use {{}} double braces to surround AngularJS expressions that you want inserted onto your page. This is called interpolation.

Use the ng-bind attribute to one-way bind the contents of a tag to a variable. Use the ng-model attribute to two-way bind a variable with an input.
