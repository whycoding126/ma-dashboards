The <ma-point-list> directive displays a list of data points which you can select from. All data points added to your Mango system will display by defualt.

You can optionally supply the query attribute to filter the list of points using RQL.

The <ma-filtering-point-list> directive allows you to select a data point using fuzzy searching by device name or point name. Search results will update as you type.
