To create your own dashboard, copy one of the templates into a new folder in ${MA_HOME}/overrides/web/modules/dashboards/web and rename the HTML file to index.html.

There is also a complete Material Design admin template similar to these documentation pages, with a menu and toolbar which you can customise.

Once you have created a folder in overrides, your dashboard will be accessible at /modules/dashboards/web/DASHBOARD_NAME/. You can change the Custom Dashboards link on the Mango menu bar to point to your new dashboard in the System settings.

Once you have a working template you can starting playing around with the live examples and copy and paste them into your template HTML file.
