JsonStoreEventManager
service in module maServices

Description

Provides an EventManager factory pointing to the json-data websocket endpoint at '/rest/v1/websocket/json-data'

  • All methods available to EventManager are available.
  • Used by <ma-json-store> directive.

Usage

    jsonStoreEventManager.unsubscribe($scope.item.xid, SUBSCRIPTION_TYPES, websocketHandler);
