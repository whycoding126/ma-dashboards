maGetPointValue
service in module maDashboards

Description

<ma-get-point-value point="myPoint"></ma-get-point-value>

  • This directive pulls the live value of a data point and outputs it onto the point object.
  • You can use the point-xid property or pass in a point from <ma-point-list>.
  • Live values can be displayed as text within your HTML by using {{myPoint.value}} or {{myPoint.renderedValue}} expressions.
  • Additionally, you can use the outputted value to make custom meters. View Example.
  • View Demo / View point-xid Demo

Usage

<md-input-container class="md-block">
    <label>Choose a point</label>
    <ma-point-list ng-model="myPoint1"></ma-point-list>
</md-input-container>
<ma-get-point-value point="myPoint1"></ma-get-point-value>
<p>Point name is "{{myPoint1.name}}" and its value is {{myPoint1.renderedValue}}.</p>

<ma-get-point-value point-xid="DP_698831" point="myPoint2"></ma-get-point-value>
<p>Point name is "{{myPoint2.name}}" and its value is {{myPoint2.renderedValue}}.</p>

Parameters

ParamTypeDetails
pointobject

The point object that the live value will be outputted to. If point-xid is used this will be a new variable for the point object. If the point object is passed into this attribute from <ma-point-list> then the point object will be extended with the live updating value.

point-xid
(optional)
string

If used you can hard code in a data point's xid to get its live values.

points
(optional)
array

Rather then passing in a single point object to point you can pass in an array of point objects (from <ma-point-query> for example) and have the live values added to each point object in the array.
