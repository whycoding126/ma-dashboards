moment
service in module maFilters

Description

Converts a timetamp to a momentJS object that can be formatted and manipulated.

  • Moment methods can be called using the filter syntax to call functions:
    • Three days from now is {{'now' | moment:'add':3:'days' | moment:'format':'LLL'}}
    • {{myPoint.time | moment:'format':'ll LTS'}}
  • View Filters Demo

Usage

In HTML Template Binding

{{ moment_expression | moment }}

In JavaScript

$filter('moment')()
