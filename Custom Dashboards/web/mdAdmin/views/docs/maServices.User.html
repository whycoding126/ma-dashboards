User
service in module maServices

Description

Provides a service for getting list of users from the Mango system, as well as logging users in and out.

  • All methods return $resource objects that can call the following methods availble to those objects:
    • $save
    • $remove
    • $delete
    • $get

Usage

 var user = User.login({
    username: $scope.username,
    password: $scope.password
});

User.logout();
