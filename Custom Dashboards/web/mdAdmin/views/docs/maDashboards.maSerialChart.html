maSerialChart
service in module maDashboards

Description

<ma-serial-chart style="height: 300px; width: 100%" series-1-values="point1Values" series-1-point="point1" default-type="column"> </ma-serial-chart>

  • The <ma-serial-chart> directive allows you to create line and bar charts.
  • Many different variations on a chart can be created by customizing the attributes.
  • Values are provided via <ma-point-values>. You can provide your time range and rollup settings to <ma-point-values>, then pass the values data to <ma-serial-chart>.
  • In the attributes starting with series... you will replace X with the number of that series starting with 1.
  • Note, you will need to set a width and height on the element.

Stacking mode of the axis. Possible values are: "none", "regular", "100%", "3d".

string

The type of chart used for all graphs. The possible values for chart type are "line", "smoothedLine", "column", or "step".

(optional)
string

The default color used for all graphs. Can be a color string or hex value.

default-axis
(optional)

The defaults axis used for all graphs. Can be "right" or "left".

(optional)
string

Overides the balloon text with a specified string.
