Point
service in module maServices

Description

Provides service for getting and and updating a list of points.

  • Used by <ma-point-list> and <ma-filtering-point-list> directives.
  • All methods return $resource objects that can call the following methods availble to those objects:
    • $save
    • $remove
    • $delete
    • $get

Usage

 Point.rql({query: 'limit(1)'}).$promise.then(function(item) {
    $scope.ngModel = item[0];
});
